# 🤖 Gespa Chat Real Avatar

Plugin WordPress de chat flutuante interativo desenvolvido para o site **gespensa.pt**, com sistema de FAQ pré-definido e interface moderna.

---

Funcionalidades

Chat flutuante ativado por botão
Sistema de FAQ com perguntas e respostas pré-definidas
Interface moderna, limpa e responsiva
Animações suaves para melhor experiência do utilizador
Integração com link de agendamento (Google Calendar)
 Respostas automáticas baseadas em lógica simples (sem IA)

---

Como funciona

O plugin funciona com base num sistema de **FAQ estático**:

- As perguntas estão definidas diretamente no código JavaScript
- Cada pergunta tem uma resposta associada
- O utilizador seleciona uma pergunta
- O sistema devolve a resposta automaticamente

Este plugin **não utiliza inteligência artificial**, apenas lógica pré-definida.
